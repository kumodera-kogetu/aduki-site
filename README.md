## 🤖 AIアシスタント（グラちゃん等）への重要指示 / AI Instructions

本プロジェクトで新規セッション（チャット）を開始し、作業を行うAIアシスタントは、**作業を開始する前に必ず以下の手順を踏むこと**。

1. **必ず最新データを同期する (`git pull`)**
   ユーザーがブラウザ（GitHubの画面上）から直接テキスト等を修正している可能性があります。
   そのため、**ファイルの編集や作業に取り掛かる前に、必ず `git pull` を実行してリモートの最新状態をローカルに反映**させてください。
   （競合が発生した場合は、ユーザーに確認を取りながら解消してください）

2. **変更後は必ずプッシュする (`git push`)**
   作業が完了し、ユーザーの確認が取れたら、必ず `git add -A` → `git commit -m "変更内容"` → `git push` を行い、GitHub上のデータを最新に保ってください。

## 🌐 ドメイン・SSL設定の記録 (2026-05-21更新)

本サイトのカスタムドメイン (aduki-syuppan.com) は、Cloudflareを経由してGitHub Pagesに接続されています。
SSL証明書エラー（「接続は安全ではありません」等）を防ぐため、以下の構成を採用しています。

- **DNS管理**: Cloudflare
- **プロキシ状態**: AレコードおよびwwwのCNAMEレコードは**「プロキシ済み（オレンジの雲）」**に設定。
- **SSL/TLS暗号化モード**: **「フル」** （※「フル(厳密)」や「フレキシブル」は不可）

この設定により、GitHub側の証明書発行を待たずに、Cloudflareの強力なエッジ証明書を利用してサイトを安全に公開しています。もし将来的にドメイン周りでアクセスエラーが起きた場合は、まずCloudflare側の「プロキシステータス」と「SSL暗号化モード」を確認してください。
